IT1x25 Web Development Project

GenLink – Case Study

A web platform designed to encourage meaningful and ongoing interactions between youths and seniors.

Project Overview

GenLink is a web-based platform that helps youths and seniors build connections through ongoing interactions. Users can create accounts, find suggested connections, manage requests, explore events, and communicate through the system.

My Role & Responsibilities

I contributed to the design and development of key features, focusing mainly on the Connections experience. This included user search and filtering, managing connection requests (accept/reject), and ensuring the UI was clear and easy to use.

Tools Used
  • HTML / CSS / JavaScript
  • Bootstrap (UI)
  • Flask (Back-end)
  • MySQL (Database)

Development Process

  1. Planning: Identified user needs and the purpose of the platform.
  2. Design: Planned pages and user flow for connections, events, and messaging.
  3. Build: Implemented features using Flask + MySQL, then designed UI with Bootstrap.
  4. Testing: Checked forms, database updates, and user experience for common actions.

Key Features

Login & Registration

Allows users to securely create and access accounts.

Homepage (Announcements + Events)

Shows announcements, upcoming events, and quick previews.

Connections + Suggested Users

Users can find people, send requests, and manage connections.

Messaging / Interaction

Supports communication to encourage ongoing engagement.
